\subsection{Detailed Description}
Class to read and write data from and to H\+D\+F5 files. 

The objects of this class can be made using pointer to a scalar field object. Once the rw\+H\+D\+F5 object is created, the member functions calls make it easy to read and write data.

\hyperlink{classfileio_ad225b147a6799b6adaaac5745ff54e1a}{write\+H\+D\+F5()} writes the current data stored in scalar field in the dataset name passed. \hyperlink{classfileio_ad87cac9f787b843aee05c312cec90e59}{read\+H\+D\+F5()} reads the dataset name passed and writes in scalar field object.
