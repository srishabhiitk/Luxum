\subsection{Detailed Description}
Class for all the global variables and functions related to parallelization. 

After M\+P\+I\+\_\+\+Init, every process has its own rank. Moreover, after performing domain decomposition, each process has its own x\+Rank and y\+Rank to identify its position within the global computational domain. These data, along with the data to identify the neighbouring processes for inter-\/domain communication are stored in the {\bfseries parallel} class. This class is initialized only once at the start of the solver.
